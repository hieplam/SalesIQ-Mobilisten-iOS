Mobilisten iOS SDK
=================

Power up your mobile app with Mobilisten.

## Changelog
Bug fix for email transcript not working when both FAQ and conversation is off. Built using Swift 5.2.
## Installation
The Mobilisten SalesIQ iOS SDK can be integrated with your application by adding the following to your podfile:

```ruby
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '9.0'

target 'Project Target' do
  use_frameworks!
  pod 'Mobilisten', '1.1.5'
end
```
